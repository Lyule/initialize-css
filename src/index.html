Initialize.css

Initialize helps you cross-browser best practices. Available in CSS and configurable SCSS format.

Download Fork on Github

Initialize.css is a complete collection of CSS best practices based on Normalize.css and HTML5 Boilerplate.

Usage

npm
npm install initialize-css

sass / scss
@import: "initialize-css";
